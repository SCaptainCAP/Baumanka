[] - масив типа
[N] - массив из N элементов типа
(type) - ф-ция, принимающая аргумент типа type и возвращающая
* - указатель на



int *(*x[10])(void)

x - это массив из 10 элементов типа указатель на функцию, ничего не принимающую и возвращающую указатель на int


char *(*(**foo[][8])())[];

foo - это массив типа массив из 8-ми элементов типа указатель на указатель на функцию, ничего не принимающую, и возвращающую указатель на массив типа указатель на char
